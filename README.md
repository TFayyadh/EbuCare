# EbuCare

A Flutter project.

## Introduction
EbuCare is a mobile application designed to support new mothers during the
postpartum period. In this application, we balance it out by putting in insights from the traditional perspective and the modern medical perspective.
The objective is to help new mothers going through this tough period with ease.
